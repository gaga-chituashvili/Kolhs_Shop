# 🛍 Kolh's Shop

Kolh's Shop is a modern **React e-commerce frontend application** built with a component-based architecture.

The project demonstrates best practices for:

* scalable React project structure
* protected routes
* authentication flow
* state management using Context API
* reusable UI components

---

# 🚀 Features

* Product listing
* Product search
* Shopping cart sidebar
* Authentication (Sign In / Sign Up)
* Protected routes
* Responsive UI
* Scroll utilities
* Component-based architecture

---

# 🧰 Tech Stack

Frontend

* React 18
* React Router DOM
* Tailwind CSS
* Lucide React Icons
* React Spinners

State Management

* React Context API
* Reducer pattern

Authentication

* JWT decoding

Development Tools

* ESLint
* Prettier
* PostCSS
* Autoprefixer

# 📁 Project Structure

```text
src
│
├── api                # API requests (authentication, products)
│
├── component          # Reusable UI components
│   ├── header
│   ├── footer
│   ├── productcard
│   ├── forms
│
├── config             # Route configuration
│
├── constant           # Application constants
│
├── context            # Global state management (Context + Reducer)
│
├── guard              # Route protection components
│
├── pages              # Application pages
│
├── utils              # Utility helpers (JWT, helpers)
│
├── App.js             # Root component
├── Approutes.jsx      # Routing structure
└── index.js           # React entry point
```

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/gaga-chituashvili/Kolhs_Shop.git
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm start
```

Build for production

```bash
npm run build
```

---

# 🧪 Code Quality

Run ESLint

```bash
npm run lint
```

Fix lint issues

```bash
npm run lint:fix
```

Format code

```bash
npm run format
```
